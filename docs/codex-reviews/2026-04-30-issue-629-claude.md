## Findings

### 1. Writer-side placeholder validation gap (Low)

**Files**: `scripts/model-fallback-log.sh` lines 73-75 vs `.github/scripts/check_fallback_log_schema.py` lines 78-83

The checker now correctly rejects placeholders on all base required string fields (`primary_model`, `fallback_model`, `caller_script`, `session_id`). The writer, however, only validates non-empty for `--primary`, `--fallback`, and `--caller` — it does not reject `--primary TODO`. The PDCAR's writer/checker parity contract implies both should enforce the same rules.

**Mitigated by**: CI checker catches it before merge. Defense-in-depth is acceptable for this slice. The parent issue (`#612`) should track whether writer-side placeholder rejection is warranted.

### 2. Writer heredoc YAML injection (Low — carried forward)

**File**: `scripts/model-fallback-log.sh` lines 124-145

The `cat <<EOF` heredoc expands variables without YAML quoting. Values containing `: `, `#`, or leading `[`/`{` could produce malformed YAML. Current callers pass safe strings. Residual risk accepted for this slice.

### 3. Independent placeholder sets (Low — carried forward)

**Files**: `scripts/model-fallback-log.sh` line 23, `.github/scripts/check_fallback_log_schema.py` line 15

Writer shell `is_placeholder` and checker Python `PLACEHOLDER_VALUES` are defined independently but currently match. A change to one without the other would create a silent parity drift. Acceptable for this slice; parent issue should track.

### 4. Redundant error on missing date (Informational)

**File**: `.github/scripts/check_fallback_log_schema.py` lines 54-63

If `date` is absent, both the "missing required fields" check and the date-format regex check fire, producing two errors for the same root cause. Not a defect, only slightly noisy output.

### 5. Unpinned PyYAML (Low — carried forward)

**File**: `.github/workflows/check-fallback-log-schema.yml` line 33

`pip install --quiet pyyaml` without a version pin. Standard for governance gates but a future PyYAML breaking change could cause silent drift.
