# Effectiveness Baseline

Tracked weekly governance metrics snapshots live here.

- `YYYY-MM-DD.json` / `YYYY-MM-DD.md`: immutable dated snapshots
- `latest.json` / `latest.md`: most recent generated baseline

Current baseline metrics:
- bug rate per repo per week
- revert rate per repo per week
- CI pass rate per repo over the last 14 days

Source of truth:
- generated by `scripts/overlord/build_effectiveness_metrics.py`
- refreshed by `.github/workflows/overlord-sweep.yml`
